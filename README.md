## 一、术语约定

- **周期**：`5h` = 5 小时 | `w` = 周 | `mo` = 月
- **额度倍率**：当前周期内可消耗的 Token 额度价值 ÷ 订阅月费。倍率越高性价比越高（前提是认可该模型的单位定价）
- **TPS**：Tokens Per Second，模型输出速度
- **价格**：未特别标注币种则为美元（$），标注 ￥ 为人民币
